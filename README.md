## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bradroehrig/Group_Project_Team_Lexi.git
   ```
2. Install Django
   ```sh
   pip install Django==2.2.4
   ```
